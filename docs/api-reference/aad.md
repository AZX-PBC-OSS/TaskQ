# Microsoft Entra ID (AAD)

`EntraIdProvider` / `EntraIdPgProvider` / `EntraIdRedisProvider` and the raw
token fetchers, backed by `azure.identity` / `azure.identity.aio`. Part of the
`taskq[aad]` extra (includes `aiohttp`). See the
[Managed Identities guide](../guides/managed-identities.md) for prerequisites
and a full worker example.

::: taskq.aad
